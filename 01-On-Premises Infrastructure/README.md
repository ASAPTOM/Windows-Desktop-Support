# Operating System

## Definition

An **Operating System (OS)** is system software that acts as a **mediator (interface)** between the **user** and the **computer hardware**. It manages hardware resources, runs applications, and allows users to interact with the computer.

## Simple Interview Answer

> **Operating System is a mediator between the end user and the computer hardware. It manages the computer's resources and provides an interface for users to run applications.**

## Example

**User** → Opens Google Chrome

**Operating System (Windows)** → Receives the request and manages the required resources.

**Hardware** → CPU, RAM, Storage, Display perform the task.

## Flow

```text
End User ⇄ Operating System ⇄ Computer Hardware
```
